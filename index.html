Google Sheet Setup Guide

  1. Go to sheets.google.com and create a new blank spreadsheet.
  2. Click Extensions → Apps Script.
  3. Delete the placeholder code, then click Copy Script below and paste it in.
  4. Save (Ctrl/Cmd+S).
  5. Deploy → New deployment → gear icon → Web app → Execute as: Me → Who has access: Anyone → Deploy.
  6. Authorize when prompted (Advanced → Go to project (unsafe) is normal for your own scripts).
  7. Copy the URL ending in /exec and paste it into the Multi-Location Sync panel, then click Save & Connect.
